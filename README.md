# DisgRAM - Modern Messenger

Месенджер нового покоління, який поєднує найкращі функції Discord та Telegram.

## 🎯 Ключові особливості

### Від Telegram:
- 🔒 Секретні чати з шифруванням E2E
- ⚡ Швидка синхронізація між пристроями
- 🤖 Потужний API для ботів
- 📢 Канали для масових розсилок
- ⏱️ Самознищувані повідомлення
- ☁️ Необмежений cloud storage

### Від Discord:
- 🎙️ Голосові та відео канали
- 🏰 Сервери з категоріями та каналами
- 👥 Система ролей та дозволів
- 🎮 Інтеграції для геймерів
- 🧵 Threads (гілки обговорень)
- 🖥️ Screen sharing

### Унікальні функції:
- 🎨 Гнучка кастомізація інтерфейсу
- 🌓 Темна/світла тема
- 📱 Кросплатформенність
- 🔔 Смарт-нотифікації
- 🔍 Потужний пошук

## 🛠️ Технології

- **Frontend**: React + TypeScript
- **Desktop**: Electron
- **State Management**: Redux Toolkit
- **UI**: Material-UI / Custom Components
- **Real-time**: Socket.io / WebRTC
- **Backend** (майбутнє): Node.js + Express + PostgreSQL

## 🚀 Швидкий старт

```bash
# Встановити залежності
npm install

# Запустити в режимі розробки
npm run dev

# Зібрати для продакшену
npm run build
```

## 📁 Структура проекту

```
disgram/
├── src/
│   ├── main/              # Electron main process
│   ├── renderer/          # React UI
│   │   ├── components/    # React компоненти
│   │   ├── features/      # Функції (чати, сервери, voice)
│   │   ├── store/         # Redux store
│   │   └── styles/        # Стилі
│   ├── shared/            # Спільний код
│   └── types/             # TypeScript типи
├── public/                # Статичні файли
└── assets/                # Іконки, зображення
```
