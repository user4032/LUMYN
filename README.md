# LUMYN

LUMYN — сучасний десктопний месенджер на Electron + React.

## Можливості
- Приватні чати та сервери (канали, ролі, учасники)
- Реальний час (Socket.IO)
- Threads‑стрічка: пости та підписки
- Профіль: банер, рамка, статус, кастомізація
- Форматування повідомлень як у Discord
- Автооновлення через GitHub Releases (electron-updater)

## Технології
Electron, React, TypeScript, MUI, Redux Toolkit, Express, Socket.IO

## Запуск локально
npm install
npm run dev
## Збірка
npm run build
